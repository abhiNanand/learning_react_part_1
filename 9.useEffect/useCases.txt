## **3ï¸âƒ£ Listening to Window Resize Events**
ðŸ‘‰ Detecting window size changes and updating state accordingly.  
```jsx
import { useState, useEffect } from "react";

export default function App() {
  const [width, setWidth] = useState(window.innerWidth);

  useEffect(() => {
    const handleResize = () => setWidth(window.innerWidth);
    
    window.addEventListener("resize", handleResize);

    return () => {
      window.removeEventListener("resize", handleResize); // Cleanup on unmount
    };
  }, []);

  return <h1>Window Width: {width}px</h1>;
}
```
**ðŸ›  How it works:**  
- Updates the width state **whenever** the window is resized.  
- **Cleans up** event listener when the component unmounts.

## **4ï¸âƒ£ Using useEffect with setInterval (Auto Counter)**
ðŸ‘‰ Runs an effect at a fixed interval.  
```jsx
import { useState, useEffect } from "react";

export default function App() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    const interval = setInterval(() => {
      setCount((prevCount) => prevCount + 1);
    }, 1000); // Increments count every second

    return () => clearInterval(interval); // Cleanup on unmount
  }, []);

  return <h1>Counter: {count}</h1>;
}
```
**ðŸ›  How it works:**  
- Starts a counter **when component mounts**.  
- **Cleans up** interval on unmount to prevent memory leaks.

## **5ï¸âƒ£ Listening to Keyboard Events**
ðŸ‘‰ Detecting a key press and updating the state.

**ðŸ›  How it works:**  
- Updates `key` whenever a key is pressed.  
- **Cleans up** the event listener when the component unmounts.

### **ðŸ’¡ Key Takeaways from useEffect**
âœ” **Without dependency array (`useEffect(() => {...})`)** â†’ Runs **after every render**.  
âœ” **With empty dependency array (`useEffect(() => {...}, [])`)** â†’ Runs **only once** (on mount).  
âœ” **With dependencies (`useEffect(() => {...}, [state])`)** â†’ Runs **whenever that state/prop changes**.  
âœ” **Cleanup function (`return () => {...}`)** â†’ Removes event listeners, intervals, etc., **to prevent memory leaks**.
